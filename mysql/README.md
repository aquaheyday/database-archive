# 🐬 MySQL Archive

이 폴더는 **MySQL 학습 및 실습 자료**를 보관하는 공간입니다.  
기본 개념, SQL 문법, 인덱스 및 성능 최적화, 예제 코드, 실습 프로젝트 등을 단계적으로 정리하며,  
학습하면서 새롭게 알게 된 내용과 트러블슈팅 경험도 함께 기록합니다.
